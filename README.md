# Ring Set Mode

A tiny script to set a location mode for Ring

## Prerequisites

- [Ring Refresh Token](https://github.com/dgreif/ring/wiki/Refresh-Tokens)

## Installation

```shell
git clone git@github.com:adhamu/ring-set-mode.git
cd ring-set-mode
fnm use
yarn
```

## Set Mode

### Home

```shell
yarn home
```

### Away

```shell
yarn away
```

### Unarmed

```shell
yarn unarmed
```
